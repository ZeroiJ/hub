# Hub - AI Development TUI

A terminal-based unified workspace for AI-assisted development, combining git management, AI coding assistants, and shell access.

## Features

- **Three-Panel Layout**: Git/Folders, AI Assistant, Shell/Dev
- **AI Integration**: Support for Claude Code, Gemini CLI, OpenCode CLI
- **Git Workflow**: Simplified commit process with AI-generated messages
- **Customizable**: TOML configuration for keybindings, themes, and more

## Installation

```bash
pip install hub-tui
```

## Usage

```bash
hub
```
